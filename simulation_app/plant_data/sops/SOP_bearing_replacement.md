## 1. PURPOSE
To establish a safe and standardized procedure for replacing bearings on rotating equipment in the plant, ensuring proper installation, alignment, and documentation.

## 2. SCOPE
This SOP applies to all bearing replacements on production machinery including motors, gearboxes, arbors, rollers, and pump shafts.

## 3. SAFETY PRECAUTIONS
- ✅ LOTO **MUST** be applied before starting any bearing work (refer to LOTO SOP)
- ✅ Minimum 2 persons required for bearing replacement
- ✅ PPE Required: Safety helmet, safety shoes, safety glasses, leather gloves
- ✅ Use only authorized bearing puller and press tools — no hammers on bearings
- ✅ Clean work area — no loose tools or debris near open bearing housings

## 4. REQUIRED TOOLS & MATERIALS

| Tool / Material | Specification |
|----------------|--------------|
| Bearing puller (mechanical or hydraulic) | SKF TMHP 10E or equivalent |
| Bearing heater (induction type) | SKF TIH 030m or equivalent |
| Torque wrench | 10-200 Nm range |
| Dial indicator with magnetic base | 0.01mm resolution |

| New bearing(s) | As per machine manual (verify part number before starting) |

| Cleaning solvent | Kerosene or SKF bearing cleaner |
| Clean lint-free rags | — |
| Feeler gauge set | 0.02 – 1.0 mm |

## 5. PROCEDURE

### Step 1: Preparation (15 min)
1. Obtain work order (WO) from shift supervisor
2. Verify bearing part number against machine manual
3. Collect new bearing from spare parts store (verify packaging seal is intact)
4. Gather all tools
5. Inform control room that machine will be taken offline

### Step 2: Isolation & LOTO (10 min)
1. Stop machine using normal controls
2. Apply LOTO on all energy sources (electrical, hydraulic, pneumatic)
3. Verify zero energy state — attempt restart
4. Tag out with name, date, time, reason

### Step 3: Access & Disassembly (30-60 min depending on machine)
1. Remove safety guards and covers
2. Disconnect coupling / belt / chain as applicable
3. Remove bearing housing cover or end plate
4. Record bearing condition before removal:
   - Photograph bearing for failure analysis
   - Note: Was bearing loose/tight on shaft? Was it noisy? Discolored?
   - Save removed bearing for root cause analysis if failure was unexpected

### Step 4: Old Bearing Removal (15-30 min)
1. Mark shaft position for reference
2. Attach bearing puller — grip on inner race (never pull by outer race)
3. Apply steady, even force — no sudden impacts
4. Extract bearing completely
5. Inspect shaft:
   - Check for scoring, wear, or damage
   - Measure shaft diameter at bearing seat (must be within tolerance per manual)
   - Clean shaft with solvent and lint-free rag

### Step 5: New Bearing Installation (15-30 min)
1. Inspect new bearing — rotate by hand, must be smooth with no roughness
2. **Heating method (recommended for interference fit):**
   - Place bearing on induction heater
   - Heat to 80-100°C (bearing will expand for easy mounting)
   - Slide bearing onto shaft quickly — push against shoulder
   - Allow to cool naturally (do NOT quench with water)
3. **Press method (alternative):**
   - Use arbor press with correct-size sleeve on inner race
   - Apply force evenly — no tilting
   - Press until bearing seats against shoulder
4. Apply grease:
   - Fill bearing with grease (1/3 to 1/2 full cavity)
   - Use correct grease type per lubrication chart
   - Apply grease to bearing housing as well

### Step 6: Reassembly (30-60 min)
1. Replace bearing housing cover / end plate
2. Reconnect coupling / belt / chain
3. Check shaft end float with dial indicator (must be within manual specification)
4. If motor bearing: Perform laser alignment
   - Acceptable tolerance: 0.05mm offset, 0.05mm/100mm angular
5. Replace safety guards

### Step 7: Testing (15-30 min)
1. Remove LOTO
2. Start machine at no load (jog mode if available)
3. Listen for abnormal noise
4. Measure vibration: **must be below 2.5 mm/s RMS on new bearing**
5. Monitor bearing temperature for 15 minutes:
   - New bearing may run warm initially (up to 70°C)
   - Should stabilize below 60°C within 30 minutes
6. If all OK — release machine for production

### Step 8: Documentation (10 min)
1. Complete work order with:
   - Parts used (bearing P/N, quantity)
   - Labor hours
   - Vibration and temperature readings (post-installation)
   - Observations / photos
2. Update machine maintenance log
3. Update spare parts inventory (deduct used parts)
4. If bearing failed prematurely — initiate Root Cause Analysis (RCA)
5. Return old bearing to maintenance workshop for analysis

## 6. QUALITY CHECKS

| Check Point | Accept Criteria | Tool |
|------------|----------------|------|
| Shaft diameter at bearing seat | Within tolerance (typically h6) | Micrometer |
| Bearing fit on shaft | Tight (cannot slide freely) | Hand feel + feeler gauge |
| Post-installation vibration | < 2.5 mm/s RMS | Vibration meter |
| Post-installation temperature | Stabilizes < 60°C in 30 min | IR thermometer |
| Shaft alignment (if motor) | < 0.05mm offset | Laser alignment tool |
| End float | Per manual (typically 0.05-0.15mm) | Dial indicator |

## 7. COMMON MISTAKES TO AVOID
- ❌ Hammering bearing onto shaft — causes brinelling damage
- ❌ Pulling by outer race — damages rolling elements
- ❌ Using old grease or wrong grease type — causes premature failure
- ❌ Skipping alignment after motor bearing change — causes coupling vibration
- ❌ Over-greasing — causes overheating
- ❌ Not recording post-installation vibration — no baseline for future monitoring
- ❌ Re-using seals or O-rings — always replace with new
